# Bước 1: Dọn Tài Liệu Và Cấu Trúc

## Trạng Thái

Đang thực hiện.

## Mục Tiêu

Làm tài liệu ngắn, đúng, không bị trùng lặp và không mô tả cấu trúc cũ.

## Checklist

- Gom Markdown vào `docs/`.
- Xóa các file Markdown cũ ở root.
- Viết lại tổng quan dự án trong `docs/README.md`.
- Tách kế hoạch triển khai theo từng bước trong `docs/implementation/`.
- Cập nhật hướng dẫn agent trong `docs/AGENT_GUIDE.md`.
- Loại bỏ mô tả file cũ như `feed.js`, `home.js`, `document.js`, `search.js`, `notifications.js`, `css/style.css`.
